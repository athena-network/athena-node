Use free public nodes that are load balanced with high-availability with the wallet quickly and safely.

Instructions

Step 1 Open a command line window and navigate to your Athena directory.

Step 2 Run the below command that is relevant to your system and version.


./wallet --remote-daemon public.athxnode.io

To use nodes in a specific region, use one of the following hostnames:

us-west athxnode.io
us-east athxnode.io
europe athxnode.io
asia athxnode.io
